Re:Cycle

A game jam project made in Unity, built around the theme of "Sustainability Hunt". The game is a 3D game where the player aims to combine resources that are given, in order to create self sustaining systems. The game is still a work in progress, but the core mechanics are in place and it's a fun little project to tinker with. Currently not being developed, although plans to restart development are in place. The game was made for the Volvo Game Jam 2025.

A fast-paced, action-oriented zombie shooter built in Unity with a focus on smooth gameplay and engaging combat mechanics.
